<p>This project is my personal social network; it features logic similar to that of any social network, including real-time chat, posts, user profiles, profile and cover image uploads, online status and messaging, friend requests, and authentication via email and Google.
The project was developed using React, Vite, TypeScript, Tailwind, Node.js, and Firebase, with Vercel used for deploying the demo.</p>
